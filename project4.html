Solution

Through user research and thorough examination of the existing app, as well as the MARTA system overall, I redesigned an enhanced MARTA app experience that focuses on user needs. Instead of throwing transit information at users, the redesign allows for users to:

Easily seek assistance with medical issues

This feature makes it easy for spanish-speaking farmworkers to request for medical help through the app. Once they request for help, someone from the fictional ICFW would reach out to the user via phone.

Chatbot-based help

We designed an AI-based feature that would help the user feel cared for. This feature would help the user get the help they might be needing in a conversational manner.

Always know the next steps

We also designed a centralized location for the user to access their requests for help in the app. This feature ensures that the user isn‚Äôt left hanging ‚Äì this would be the last thing they would want amid the pandemic.
